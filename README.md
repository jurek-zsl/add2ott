- OpenTogetherTube Room selector.

- Native Youtube controls enabler for OTT player.
- Account and guest support.

## âš™ï¸How it Works

1. You enter `server_url` and optionally `login` and `password`.
2. It connects to server specified by you. You can now select room and add videos.
3. You can add videos through pop-up or with integrated button on Youtube video page.

## ðŸ“¥Installation

> [!NOTE]
> Adding to `Chrome Web Store` and `Firefox Addons` soon!

**Chrome:**

1. Download `Add2OTT-chromium.zip` from Releases.

2. Go to `chrome://extensions`.

3. Enable "Developer mode" (top right).

4. Drag & drop the `.zip` file, then select "Keep".

**Firefox:**

1. Download `Add2OTT.xpi` from Releases.

2. Go to `about:addons`.

3. Drag & drop the `.zip` file and click "Install".
